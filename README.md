# My Project Portfolio
A showcase of my favourite projects. Accessible at:

## Dev flow

### Project setup
```
yarn install
```

#### Compiles and hot-reloads for development
```
yarn serve
```

#### Compiles and minifies for production
```
yarn build
```

#### Run your unit tests
```
yarn test:unit
```

#### Lints and fixes files
```
yarn lint
```

#### Deploy to github pages
```
yarn deploy
```
